# ✨ Snappy! — Party Photobooth

A fun, playful 4-photo strip photobooth built with Next.js.

## Features
- 📸 4-photo strip with animated countdown
- 🎨 8 live camera filters (Natural, B&W, Sepia, Vivid, Warm, Cool, Dreamy, Retro)
- 🎉 Draggable emoji stickers on each photo (double-tap to remove)
- 💾 Download finished strip as a JPEG (with filters + stickers baked in)
- 🎊 Confetti celebration after shooting!
- 🪞 Mirror toggle
- 📱 Fully responsive (mobile + desktop)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## How to use
1. Allow camera access when prompted
2. Pick a filter
3. Hit **📸 Start!** — 3-second countdown before each of 4 shots
4. In review mode, tap any photo in the strip to add stickers
5. Hit **💾 Download Strip!** to save your photo strip
